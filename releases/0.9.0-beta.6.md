# Viandly 0.9.0-beta.6

Released: July 23, 2026

This closed-beta update improves recipe acquisition and maintenance by preserving more website
metadata, comparing saved recipes with their original sources, and making search and installed-app
navigation more reliable.

## Highlights

- Expanded website imports to preserve useful structured metadata including descriptions, authors,
  source names, yield, servings, preparation and cooking times, inactive time, and tags.
- Added **Check source** for one recipe and **Review website sources** for an owned recipe library.
- Added bounded background source checking with progress, cancellation, and actionable results in
  the Inbox.
- Added side-by-side source comparisons that never change a recipe until its owner applies selected
  additions or conflict resolutions.
- Added **Keep current**, **Replace with source**, and **Append source** choices for attribution
  conflicts.
- Made proposed website tags editable before application while preserving all existing recipe
  content and personal cooking data.
- Fixed live recipe searching so older responses no longer replace newer keystrokes.
- Added an installed-app Back control that restores the exact prior Viandly screen, including
  recipe search text, filters, sorting, and pagination.
- Fixed the new-cookbook form layout on phone-width screens.
- Added Help guidance for website source review and installed-app navigation.

## Compatibility

The release adds internal records for source-review batches and results. Existing recipes are not
changed automatically: source review is owner-initiated, and every proposed update requires an
explicit per-recipe action.
